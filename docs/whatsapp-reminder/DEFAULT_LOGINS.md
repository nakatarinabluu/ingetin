# 🔐 Ingetin WhatsApp Reminder - Default Login Credentials

This document contains the default credentials generated by the database seeding script (`prisma/seed.ts`). Use these to log into the [Dashboard](http://localhost:8080) for testing and development.

---

## 👑 Administrator Account
The Admin account has full access to the system, including user management, license generation, and live traffic monitoring.

* **Username:** `admin` *(or Email: `admin@ingetin.com`)*
* **Password:** `INGETIN_ROOT_SECURE_2026`

> **Note:** If you have set the `ADMIN_PASSWORD` variable in your environment files (`.env`), the system will use that password instead of the default one above.

---

## 👤 Normal User Accounts
The seed script automatically generates 50 normal test users (`user_001` through `user_050`). You can use any of these to test the standard user experience, such as linking WhatsApp, syncing Google Calendar, and creating reminders.

---

### 🔄 How to Reset or Reseed the Database
If you ever need to wipe your database and recreate these default accounts, you can run the seed script directly from the API container:

```bash
docker exec -it ingetin-apps-whatsapp-reminder-api npx prisma db seed
```
